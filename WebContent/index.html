Edge Detector beta

Reconstruction of an uploaded image by the user, broken down into the two components x and y by DFT. It is recommended to use vector image type than raster one: the former result is clearly better.

Only jpg and png files are accepted

# of Epicycles refers to the number of circles (and consequently, the number of frequency components) displayed.
Speed refers to the speed taken to draw the path.
